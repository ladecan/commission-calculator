DCD · 워런티 선택하고 저장

계약금액은 자동으로 들어와 있어요. DCD 박스 수와 워런티 연수를 +/− 버튼으로 선택하면 커미션이 자동 계산됩니다. 확인 후 "이 결과 저장하기" 버튼을 누르세요.

소모품(DCD) 선택 기준
• 기본 1박스는 차감 없음 — 추가 박스부터 차감
• 일반 딜러: 추가 1박스당 75만원 차감
• 휴먼홀딩스: 기본 2박스까지 차감 없음, 추가부터 누적 차감
워런티 선택 기준
• GMPP 0~1년: 차감 없음
• GMPP 2년 (Add 1년): 900만원 차감
• GMPP 3년 (Add 2년): 1,400만원 차감
• VBP 3년: 26년~ 납품 2,100만 / 25년 이전 2,000만 차감
• VBP 4년: 26년~ 납품 2,700만 / 25년 이전 2,600만 차감
💡 VBP 3년 이상 선택 시 납품 연도 토글이 자동으로 나타나요

저장 버튼 꼭 누르기

계산만 하고 저장 안 하면 관리자에게 안 보여요. 반드시 "저장하기" 버튼까지 눌러주세요!

자주 묻는 질문
내 이름이 없어요
이번 달 데이터가 아직 업로드되지 않았거나 이름이 다르게 등록됐을 수 있어요. 관리자에게 문의해주세요.
계약금액이 실제와 달라요
계약금액은 엑셀 데이터 기준으로 자동 입력되며 수정이 불가해요. 금액이 다르다면 관리자에게 문의해주세요.
저장했는데 다시 들어가면 없어요
정상이에요! 내 이름 → 거래처 순서로 다시 들어가면 ✓ 저장됨 표시와 함께 이전에 입력한 값이 그대로 나와요.
잘못 저장했어요
다시 같은 거래처로 들어가서 수정하고 "결과 다시 저장하기" 버튼을 누르면 덮어씌워져요.
